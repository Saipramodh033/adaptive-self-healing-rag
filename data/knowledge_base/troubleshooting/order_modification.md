# Order Modification
Source: troubleshooting/order_modification.md

**Last Updated:** January 2025
**Applies to:** All orders on ShopEase.com and the ShopEase mobile app

## Modifying the Delivery Address
You can change the delivery address of an order **only before it has shipped**.
1. Go to **My Orders**.
2. Select the order.
3. Click **Change Shipping Address** and select a new saved address.

If the order has already shipped, the address cannot be modified.

## Changing Item Quantity
We cannot modify the quantity of items in an existing order. 
If you need more or fewer items, you must **cancel the original order** (if it hasn't shipped yet) and place a new order.

## Item Substitution (Color/Size)
We do not support item substitution. If you selected the wrong color or size, please cancel the order before shipment and reorder. 

## Timeline for Modifications
For Same-Day or Express Delivery orders, our fulfillment process is extremely fast. You must make any address modifications or cancellations within **1 hour of placement**. After 1 hour, the order is locked for transit.

## Contact Information
